# 清除缓存方法

本项目使用 Vite + React，以下是开发过程中遇到卡顿或异常时的缓存清除方法。

## 1. 清除 Vite 缓存（最常用）

```bash
rm -rf node_modules/.vite
```

然后重新启动开发服务器：

```bash
npm run dev
```

## 2. 重装依赖（问题较严重时）

```bash
rm -rf node_modules
npm install
npm run dev
```

## 3. 增加 Node.js 内存限制（项目变大后卡顿）

```bash
NODE_OPTIONS="--max-old-space-size=4096" npm run dev
```

## 什么时候需要清除缓存？

- 开发服务器运行一段时间后明显变慢
- 修改了 `vite.config.js` 但没有生效
- 样式或组件更新后浏览器显示异常
- HMR 热更新失效，需要手动刷新才能看到变化
